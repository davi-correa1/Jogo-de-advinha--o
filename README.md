# __Jogo de Adivinhação.__
  ### _[Link do jogo aqui.](https://jogo-de-adivinhacao-one.vercel.app/)_
***
* ## Como jogar
    * Você terá 10 tentativas para adivinhar qual foi o número gerado, aleatóriamente, entre 0 e 100.
    * Se o 'número digitado' for menor que o 'número gerado', você será informado de que o 'numero digitado' é baixo.
    * Se o 'numero digitado' for maior que o 'numero gerado', você será informado de que o 'numero digitado' é maior.
    * Se você zerar suas tentativas, é _**Game Over!**_

* ## Descrição do Jogo
    * Este jogo de navegador foi meu _**primeiro projeto**_, criado com o _propósito de praticar minhas habilidades e novos aprendizados_ e foi desenvolvido por um _adolescente de 15 anos, sozinho, em 2025 com apenas alguns videos no YouTube_.
